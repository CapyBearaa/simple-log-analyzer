# Simple Log Analyzer 🔐

A basic Python tool for analyzing log files and detecting suspicious activity patterns.

## 📌 Description
This project is a simple implementation of a log analysis tool inspired by Security Operations Center (SOC) workflows.  
It scans log files and identifies potentially suspicious events such as errors, failed logins, and warnings.

## ⚙️ Features
- Detects suspicious keywords (ERROR, FAILED, WARNING)
- Simple and readable output
- Easy to modify and extend

## 🧠 Purpose
This project was created as a first step into cybersecurity, focusing on understanding how security analysts work with logs and detect anomalies.

## 🚀 How to run

```bash
python main.py
